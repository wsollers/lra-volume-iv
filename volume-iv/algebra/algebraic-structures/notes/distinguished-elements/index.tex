\section{Distinguished Elements}
\label{sec:distinguished-elements}

\algebraTopicToolkit{Distinguished Elements}{
Distinguished elements are named constants or elements selected by equations:
identities do nothing, inverses undo, absorbing elements collapse an operation,
and constants appear as part of a signature.
}
