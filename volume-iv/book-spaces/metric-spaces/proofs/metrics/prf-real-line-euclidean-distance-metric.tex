\begin{proposition*}[The Real-Line Euclidean Distance is a Metric]
The function \(d_{\mathbb{R}}(x,y)=|x-y|\) is a metric on \(\mathbb{R}\).
\end{proposition*}

\begin{proof}[Professional Standard Proof]
\LRAProofBodyStart
For all \(x,y\in\mathbb{R}\), \(d_{\mathbb{R}}(x,y)=|x-y|\geq 0\).  Moreover,
\(|x-y|=0\) if and only if \(x-y=0\), which holds if and only if \(x=y\).
Symmetry follows from
\[
  d_{\mathbb{R}}(x,y)=|x-y|=|-(y-x)|=|y-x|=d_{\mathbb{R}}(y,x).
\]
For \(x,y,z\in\mathbb{R}\), the absolute-value triangle inequality gives
\[
  d_{\mathbb{R}}(x,y)=|x-y|=|(x-z)+(z-y)|
  \leq |x-z|+|z-y|
  =d_{\mathbb{R}}(x,z)+d_{\mathbb{R}}(z,y).
\]
Thus \(d_{\mathbb{R}}\) satisfies the metric axioms.
\end{proof}

\begin{proof}[Detailed Learning Proof]
\LRAProofBodyStart
The metric axioms are checked directly.  Nonnegativity and the zero-distance
condition are properties of absolute value.  Symmetry is the identity
\(|u|=|-u|\) applied to \(u=x-y\).  The triangle inequality is the ordinary
absolute-value triangle inequality applied to the decomposition
\[
  x-y=(x-z)+(z-y).
\]
\end{proof}

\begin{remark*}[Proof structure]
The proof translates each metric axiom into a standard property of absolute
value.
\end{remark*}
