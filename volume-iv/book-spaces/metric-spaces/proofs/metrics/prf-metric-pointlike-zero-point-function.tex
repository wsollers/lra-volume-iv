\begin{theorem*}[Pointlike Functions with a Zero Are Point Functions]
Let \((X,d)\) be a metric space and let \(u:X\to\mathbb{R}_{\geq 0}\).  Then
\(u\in\delta(X)\) if and only if \(u\) is pointlike on \(X\) and
\[
  0\in u(X).
\]
\end{theorem*}
