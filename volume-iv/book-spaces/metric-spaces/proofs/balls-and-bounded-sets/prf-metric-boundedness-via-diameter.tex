\begin{theorem*}[Boundedness via Diameter]
Let \((X,d)\) be a metric space and let \(A\subseteq X\) be nonempty.  Then
\(A\) is bounded if and only if there exists \(C>0\) such that
\[
  d(a,b)\leq C
\]
for all \(a,b\in A\).
\end{theorem*}
